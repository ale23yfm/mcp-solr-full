Job Model (uniquekey: url)

Câmp Tip Obligatoriu Descriere
url string Da URL complet către pagina detaliată a jobului. Unique
title string Da Titlul exact al postului (max 200 caractere, fără HTML)
company string Nu Numele companiei (nume legal, CU DIACRITICE)
cif string Nu CIF/CUI al companiei
location string[] Nu Array de orașe/adrese din România
tags string[] Nu Tag-uri pentru skills/educație/experiență (lowercase, fără diacritice, max 20)
workmode string Nu "remote", "on-site", "hybrid"
date date Nu Timestamp UTC ISO8601 al scrape-ului
status string Nu "scraped", "tested", "published", "verified"
vdate date Nu Data verificării (ISO8601)
expirationdate date Nu Data expirării jobului (vdate + max 30 zile)
salary string Nu Format: "MIN-MAX CURRENCY" (ex: "5000-8000 RON")

Company Model (uniquekey: id)

Câmp Tip Obligatoriu Descriere
id string Da CIF/CUI (exact 8 cifre, fără prefix RO). Unique
company string Da Nume legal din Registrul Comerțului (CU DIACRITICE)
status string Nu "activ", "suspendat", "inactiv", "radiat"
location string[] Nu Array de orașe/adrese din România
website string[] Nu Site-uri oficiale ale companiei (URL valid HTTP/HTTPS)
career string[] Nu Pagini de carieră oficiale (URL valid HTTP/HTTPS)

Exemple JSON

Job

{
  "url": "https://example.com/jobs/php-developer-123",
  "title": "PHP Developer",
  "company": "Tech Corp SRL",
  "cif": "12345678",
  "location": ["București", "Cluj-Napoca"],
  "tags": ["php", "mysql", "linux", "docker"],
  "workmode": "hybrid",
  "date": "2026-01-15T10:00:00Z",
  "status": "published",
  "vdate": "2026-01-20T14:30:00Z",
  "expirationdate": "2026-02-20T14:30:00Z",
  "salary": "5000-8000 RON"
}

Company

{
  "id": "12345678",
  "company": "Tech Corp SRL",
  "status": "activ",
  "location": ["București"],
  "website": ["https://techcorp.ro"],
  "career": ["https://careers.techcorp.ro"]
}
