Acest proiect este un MCP (Model Context Protocol) Server pentru Apache Solr scris în PHP procedural. Server-ul gestionează date despre Job-uri și Companii din proiectul peviitor_core.

Caracteristici principale

  • ✅ PHP pur, fără dependențe externe
  • ✅ Protocol JSON-RPC 2.0 peste STDIO
  • ✅ Suport pentru autentificare Solr
  • ✅ CRUD complet pentru Job și Company
  • ✅ Configurare prin variabile de mediu

Structura Proiectului

mcp-solr/
├── src/
│   └── functions.php      # Toate funcțiile PHP
├── mcp-server.php         # Punct de intrare server MCP
├── Dockerfile             # Imagine Docker PHP CLI
├── opencode.json         # Configurare OpenCode
├── AGENTS.md            # Ghid pentru dezvoltatori
├── INSTRUCTIONS.md       # Instrucțiuni proiect
└── docs/                 # Documentație HTML

Quick Start

# Build Docker
docker build -t mcp-solr .

# Rulează serverul
docker run --rm -i -e SOLR_HOST=solr mcp-solr

# Testează
echo '{"jsonrpc": "2.0", "method": "job_select", "params": {}, "id": 1}' | docker run --rm -i mcp-solr
