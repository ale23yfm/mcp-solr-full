Formatul Cererilor

Toate cererile folosesc formatul JSON-RPC 2.0:

{
  "jsonrpc": "2.0",
  "method": "job_select",
  "params": {
    // parametri specifici metodei
  },
  "id": 1
}

Job API

GET job_select

Caută joburi în Solr cu opțiuni flexibile

Parametru Tip Implicit Descriere
query string *:* Query Solr
start int 0 Offset pentru paginare
rows int 10 Număr rezultate
filters array [] Filtre: status, company, workmode, location 
// Exemplu
{
  "method": "job_select",
  "params": {
    "query": "title:developer",
    "start": 0,
    "rows": 20,
    "filters": {
      "status": "published",
      "workmode": "remote"
    }
  }
}

GET job_search

Căutare simplificată cu filtre comune

Parametru Tip Descriere
term string Termen de căutare
status string Filtru status
company string Filtru companie
workmode string Filtru mod lucru
location string Filtru locație
offset int Offset
limit int Limită rezultate

POST job_insert / job_index

Adaugă job nou (url este uniquekey)

{
  "method": "job_insert",
  "params": {
    "url": "https://example.com/job/123",
    "title": "PHP Developer",
    "company": "Tech Corp",
    "cif": "12345678",
    "location": ["București"],
    "tags": ["php", "mysql"],
    "workmode": "remote",
    "status": "scraped",
    "salary": "5000-8000 RON"
  }
}

DELETE job_delete

Șterge job după URL

{
  "method": "job_delete",
  "params": {
    "url": "https://example.com/job/123"
  }
}

PUT job_update

Actualizează câmpuri existente

{
  "method": "job_update",
  "params": {
    "url": "https://example.com/job/123",
    "fields": {
      "status": "published",
      "vdate": "2026-01-20T10:00:00Z"
    }
  }
}

GET job_get

Obține job după URL

{
  "method": "job_get",
  "params": {
    "url": "https://example.com/job/123"
  }
}

Răspunsuri

Succes

{
  "jsonrpc": "2.0",
  "id": 1,
  "result": {
    // rezultatul cererii
  }
}

Eroare

{
  "jsonrpc": "2.0",
  "id": 1,
  "error": {
    "code": -32603,
    "message": "url is required"
  }
}
