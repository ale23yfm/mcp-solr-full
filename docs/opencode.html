Configurare opencode.json

Adaugă următoarea configurație în fișierul opencode.json din rădăcina proiectului:

{
  "$schema": "https://opencode.ai/config.json",
  "mcp": {
    "mcp-solr": {
      "type": "local",
      "command": ["docker", "run", "--rm", "-i", 
        "-e", "SOLR_HOST=solr", 
        "-e", "SOLR_PORT=8983", 
        "-e", "SOLR_USER=solr", 
        "-e", "SOLR_PASS=SolrRocks", 
        "mcp-solr"],
      "enabled": true
    },
    "chrome-devtools": {
      "type": "local",
      "command": ["npx", "-y", "chrome-devtools-mcp"],
      "enabled": true
    }
  },
  "command": {
    "instructions": {
      "description": "Show project instructions",
      "agent": "explore"
    }
  }
}

Comenzi Personalizate

Poți adăuga comenzi personalizate în directorul .opencode/commands/:

Exemplu: Comanda /instructions

# Creează fișierul
mkdir -p .opencode/commands

# Creează comanda
cat > .opencode/commands/instructions.md << 'EOF'
---
description: Show project instructions
agent: explore
---

Read and display INSTRUCTIONS.md and AGENTS.md from the current project.
EOF

Tool-uri Disponibile

După configurare, vei avea acces la următoarele tool-uri:

Job Tools

  • job_select - Caută joburi
  • job_search - Căutare simplă
  • job_insert / job_index - Adaugă job
  • job_delete - Șterge job
  • job_update - Actualizează job
  • job_get - Obține job

Company Tools

  • company_select - Caută companii
  • company_search - Căutare simplă
  • company_insert / company_index - Adaugă companie
  • company_delete - Șterge companie
  • company_update - Actualizează companie
  • company_get - Obține companie

Exemple de Utilizare

După configurare, poți folosi comenzi naturale în OpenCode:

Caută toate joburile publicate:
"Use job_select to find all jobs with status published"
Adaugă un job nou:
"Insert a new job with title PHP Developer, company TechCorp, url https://example.com/job1"
Obține o companie:
"Get company details for id 12345678"

Restart OpenCode

După modificarea configurației, restart OpenCode pentru a încărca noile setări.
