Introducere

MCP Solr Server funcționează prin protocolul JSON-RPC 2.0 peste STDIN/STDOUT. Trimite cereri JSON și primești răspunsuri JSON.

💡 Sfat: Serverul MCP nu are "state" persistent. Fiecare cerere este independentă.

Formatul Cererilor

O cerere JSON-RPC trebuie să conțină:

  • jsonrpc: "2.0" (versiunea protocolului)
  • method: Numele funcției de apelat
  • params: Parametrii funcției (obiect)
  • id: ID-ul cererii (opțional dar recomandat)
{
  "jsonrpc": "2.0",
  "method": "job_select",
  "params": {
    "query": "*:*",
    "start": 0,
    "rows": 10
  },
  "id": 1
}

Exemple Practice

1. Căutare Joburi

# Caută toate joburile
echo '{"jsonrpc": "2.0", "method": "job_select", "params": {}, "id": 1}' | docker run --rm -i mcp-solr

# Caută joburi cu filtru pe status
echo '{"jsonrpc": "2.0", "method": "job_select", "params": {"query": "*:*", "filters": {"status": "published"}}, "id": 1}' | docker run --rm -i mcp-solr

# Căutare simplificată
echo '{"jsonrpc": "2.0", "method": "job_search", "params": {"term": "developer", "workmode": "remote"}, "id": 1}' | docker run --rm -i mcp-solr

2. Inserare Job Nou

echo '{"jsonrpc": "2.0", "method": "job_insert", "params": {
  "url": "https://example.com/job/123",
  "title": "PHP Developer",
  "company": "Tech Corp SRL",
  "cif": "12345678",
  "location": ["București", "Cluj-Napoca"],
  "tags": ["php", "mysql", "linux"],
  "workmode": "hybrid",
  "status": "scraped",
  "salary": "5000-8000 RON"
}, "id": 1}' | docker run --rm -i mcp-solr

3. Obținere Job după URL

echo '{"jsonrpc": "2.0", "method": "job_get", "params": {"url": "https://example.com/job/123"}, "id": 1}' | docker run --rm -i mcp-solr

4. Actualizare Job

echo '{"jsonrpc": "2.0", "method": "job_update", "params": {
  "url": "https://example.com/job/123",
  "fields": {"status": "published", "vdate": "2026-01-15T10:00:00Z"}
}, "id": 1}' | docker run --rm -i mcp-solr

5. Ștergere Job

echo '{"jsonrpc": "2.0", "method": "job_delete", "params": {"url": "https://example.com/job/123"}, "id": 1}' | docker run --rm -i mcp-solr

6. Operații Companii

# Inserare companie
echo '{"jsonrpc": "2.0", "method": "company_insert", "params": {
  "id": "12345678",
  "company": "Tech Corp SRL",
  "status": "activ",
  "location": ["București"],
  "website": ["https://techcorp.ro"],
  "career": ["https://careers.techcorp.ro"]
}, "id": 1}' | docker run --rm -i mcp-solr

# Obținere companie
echo '{"jsonrpc": "2.0", "method": "company_get", "params": {"id": "12345678"}, "id": 1}' | docker run --rm -i mcp-solr

# Căutare companii
echo '{"jsonrpc": "2.0", "method": "company_select", "params": {"query": "tech", "filters": {"status": "activ"}}, "id": 1}' | docker run --rm -i mcp-solr

Exemple cu OpenCode

După configurarea OpenCode cu MCP serverul, poți folosi comenzi naturale:

Exemplu 1: "Caută toate joburile de tip PHP Developer în remote" 
/job_search term:php workmode:remote
Exemplu 2: "Adaugă un job nou pentru un dezvoltator React" 
/job_insert url:https://company.com/jobs/react-dev title:"React Developer" company:"Tech Inc" workmode:remote
Exemplu 3: "Obține detaliile companiei cu CIF 12345678" 
/company_get id:12345678

Filtre Disponibile

Job Filters

  • status: scraped, tested, published, verified
  • company: Numele companiei
  • workmode: remote, on-site, hybrid
  • location: Orașul
  • tags: Tag-uri de skills

Company Filters

  • status: activ, suspendat, inactiv, radiat
  • location: Orașul

Gestionarea Erorilor

Dacă apare o eroare, vei primi un răspuns de forma:

{
  "jsonrpc": "2.0",
  "id": 1,
  "error": {
    "code": -32603,
    "message": "url is required"
  }
}

Coduri de Eroare Comune

  • -32600: Invalid Request
  • -32601: Method not found
  • -32602: Invalid params
  • -32603: Internal error

Testare Locală

# Rulează serverul într-un terminal
php mcp-server.php

# În alt terminal, trimite cereri
echo '{"jsonrpc": "2.0", "method": "job_select", "params": {}, "id": 1}' | php mcp-server.php
Atenție: Asigură-te că Solr este accesibil înainte de a trimite cereri. Serverul va returna eroare de conexiune dacă Solr nu este disponibil.
