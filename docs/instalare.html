Cerințe

  • PHP 8.2+ (pentru rulare locală)
  • Docker (recomandat)
  • Apache Solr server (pentru conectare)

Instalare cu Docker

# Clonează proiectul
git clone <repo-url> mcp-solr
cd mcp-solr

# Build imagine
docker build -t mcp-solr .
Notă: Imaginea Docker include PHP 8.2 CLI și extensia cURL.

Instalare Locală

# Verifică versiunea PHP
php --version

# Verifică sintaxa
php -l src/functions.php
php -l mcp-server.php

# Rulează serverul
php mcp-server.php
Atenție: Pentru rulare locală, trebuie să ai extensia cURL activată în PHP.

Verificare Instalare

# Testează serverul
echo '{"jsonrpc": "2.0", "method": "job_select", "params": {}, "id": 1}' | php mcp-server.php

# Ar trebui să primești răspunsul:
# {"jsonrpc": "2.0", "id": null, "result": {"capabilities": {...}}}
