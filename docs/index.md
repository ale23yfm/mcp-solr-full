# MCP Solr Server

MCP Server for Apache Solr in PHP that handles Job and Company data from peviitor_core.

## Quick Links

- [Project Instructions](INSTRUCTIONS.md)
- [Developer Guidelines](AGENTS.md)

## Features

- Job CRUD operations via MCP
- Company CRUD operations via MCP
- JSON-RPC 2.0 over STDIO
- No external PHP dependencies
