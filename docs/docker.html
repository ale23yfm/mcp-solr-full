Build Imagine

# Build imagine Docker
docker build -t mcp-solr .

Rulare Server

# Rulează cu variabile de mediu
docker run --rm -i \
  -e SOLR_HOST=solr \
  -e SOLR_PORT=8983 \
  -e SOLR_USER=solr \
  -e SOLR_PASS=SolrRocks \
  mcp-solr
Important: Folosește întotdeauna flag-ul -i (interactive) pentru a permite citirea din STDIN.

Exemple de Rulare

Testare rapidă

# Testează conexiunea
echo '{"jsonrpc": "2.0", "method": "job_select", "params": {}, "id": 1}' | docker run --rm -i mcp-solr

Rulează în background

# Pornește containerul în detașat
docker run -d --name mcp-solr \
  -e SOLR_HOST=solr \
  mcp-solr

# Verifică logs
docker logs mcp-solr

# Oprește
docker stop mcp-solr

Conectare la rețea

# Conectare la rețeaua host-ului
docker run --network host mcp-solr

# Sau folosește numele containerului Solr
docker run --rm -i --network my-network \
  -e SOLR_HOST=solr-container \
  mcp-solr

Docker Compose

version: '3.8'

services:
  mcp-solr:
    build: .
    environment:
      - SOLR_HOST=solr
      - SOLR_PORT=8983
      - SOLR_USER=solr
      - SOLR_PASS=SolrRocks
    stdin_open: true
    tty: true

Configurare OpenCode

Pentru a folosi serverul MCP în OpenCode:

{
  "mcp": {
    "mcp-solr": {
      "type": "local",
      "command": ["docker", "run", "--rm", "-i", 
        "-e", "SOLR_HOST=solr", 
        "-e", "SOLR_PORT=8983", 
        "-e", "SOLR_USER=solr", 
        "-e", "SOLR_PASS=SolrRocks", 
        "mcp-solr"],
      "enabled": true
    }
  }
}
