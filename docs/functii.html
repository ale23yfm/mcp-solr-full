Job Operations

job_select
Caută joburi cu interogări flexibile

Parametri: query, start, rows, filters

job_search
Căutare simplificată cu filtre comune

Parametri: term, status, company, workmode, location, offset, limit

job_insert / job_index
Adaugă joburi noi în Solr (url este uniquekey)

Parametri: url (requir ed), title, company, cif, location, tags, workmode, date, status, vdate, expirationdate, salary

job_delete
Șterge job după url

Parametri: url

job_update
Actualizează câmpuri existente

Parametri: url, fields (array)

job_get
Obține un singur job după url

Parametri: url

Company Operations

company_select
Caută companii cu interogări flexibile

Parametri: query, start, rows, filters

company_search
Căutare simplificată cu filtre comune

Parametri: term, status, location, offset, limit

company_insert / company_index
Adaugă companii noi în Solr (id/CIF este uniquekey)

Parametri: id (required), company, status, location, website, career

company_delete
Șterge companie după id

Parametri: id

company_update
Actualizează câmpuri existente

Parametri: id, fields (array)

company_get
Obține o singură companie după id

Parametri: id
